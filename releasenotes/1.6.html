Release Notes - Madcow - Release 1.6

Upgrade Instructions from v1.5

Closures Removed

Madcow 1.6 removed closures for properties/spreadsheet files.

@global.currentDate=${new Date().format('dd/MM/yyyy')})

Must now be represented using the new Macro syntax for evaluation:

@global.currentDate=madcow.eval({ new Date().format('dd/MM/yyyy')})
Any other closures, whether inline or standalone, must be replaced with the madcow.eval command.

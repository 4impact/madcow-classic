Release Notes - Madcow - Version Release 1.8 - Enhancements

Bug

  • [MADCOW-217] - table.checkValue checks the first column if the specified column doesn't exist
  • [MADCOW-219] - Table Count Columns functionality is not documented
  • [MADCOW-267] - GW Plugin - VerifyEditable and VerifyNotEditable fail for radio group mapped by name
  • [MADCOW-279] - Full stops in the middle of a parameter for CountRows with arguments causes an error
  • [MADCOW-282] - Table - set value broken for Guidewire tables
  • [MADCOW-284] - header column count not working for guidewire plugin

Improvement

  • [MADCOW-232] - Improve handling of reserved characters

  • [MADCOW-278] - add notEquals operation to AbstractCount for Table Count Rows
  • [MADCOW-280] - checkValue doesn't work for select fields in tables

New Feature

  • [MADCOW-259] - GW Plugin - create plugin to count rows in table with pagination
  • [MADCOW-260] - GW Plugin - create plugin to count rows with a given criteria in a GW table, with pagination
  • [MADCOW-261] - GW Plugin - create plugin to select a row in a GW table, with pagination
  • [MADCOW-265] - GW Plugin - create plugin to check the selected value of a radio group
  • [MADCOW-266] - Allow local overrides of property files (to avoid conflict)

Story Card

  • [MADCOW-225] - Open source developer information documented
  • [MADCOW-275] - Allow basic authentication (username and password) to be used with InvokeUrl
